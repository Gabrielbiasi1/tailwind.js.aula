Estilização de sites com o Tailwind CSS

Introdução

Normalmente os sites que visitamos atualmente possuem alto grau de interatividade, e a aparência do site pode ser crucial para a retenção dos usuários.

Essa estilização costuma ser realizada em CSS e/ou Javascript, e até é possível criar nossos estilos a partir do zero, porém, é mais fácil e mais produtivo utilizarmos alguma biblioteca ou framework para tal tarefa.

Algumas das bibliotecas mais comuns para estilização são:

  • Bootstrap
  • Material Design
  • Tailwind CSS

O Tailwind CSS é uma biblioteca para estilização de sites, escrita em Javascript, que possui diversas classes prontas para utilização. Seu maior diferencial é que, ao invés de incluir a biblioteca inteira junto com o código do site, é gerado um código CSS com apenas as classes de estilos que foram utilizadas, diminuindo o tamanho do arquivo resultante.

Instalação

É possível utilizar o Tailwind com ou sem o Node.js instalado. Neste curso iremos utilizar o Node.js, então é necessário fazer o seu download e sua instalação. Para mais informações sobre o Node.js, visite seu site. Caso não queira instalar o Node.js, é necessário instalar uma CLI do Tailwind. Para mais informações, consulte este site.

Passos para a criação de um projeto:

  1. Crie um diretório onde os arquivos ficarão armazenados e navegue até ele. Por exemplo:
mkdir exemploTailwind
cd exemploTailwind
  1. Adicione o Tailwind através do npm:
npm install -D tailwindcss
  1. Crie o arquivo de configuração do Tailwind:
npx tailwindcss init
  1. Abra o diretório com o VSCode (ou outro editor de sua escolha):
code -r ./

  1. Crie uma pasta chamada src, onde ficarão nossos arquivos HTML.

  2. Abra o arquivo tailwind.config.js, e altere seu conteúdo para:

/** @type {import('tailwindcss').Config} */
module.exports = {
  content: ["./src/*.{html, js}"],
  theme: {
    extend: {},
  },
  plugins: [],
}
  1. Dentro da pasta src, crie um arquivo CSS chamado estilo.css, com o seguinte conteúdo:
@tailwind base;
@tailwind components;
@tailwind utilities;
  1. Execute o seguinte comando no terminal:
npx tailwindcss -i ./src/estilo.css -o ./dist/output.css --watch
  1. Dentro da pasta src, crie um arquivo HTML chamado index.html, com o seguinte conteúdo:
<!doctype html>
<html>
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <link href="../dist/output.css" rel="stylesheet">
</head>
<body>
    <p class="text-red-500 bg-gray-200">
        Olá mundo!
    </p>
</body>
</html>

  1. Visite o arquivo HTML no navegador para verificar o funcionamento do Tailwind.

  2. (Opcional) Instale a extensão Tailwind CSS IntelliSense se estiver utilizando o VSCode.

Tailwind Play

Também é possível, no site do Tailwind, utilizar as classes sem fazer instalação. O endereço é o https://play.tailwindcss.com/.

Utilização

Para utilizar as classes do Tailwind, basta alterar o atributo class de uma tag HTML. A seguir veremos alguns exemplos.

Tipografia

Há três tipos principais de fontes:

  • font-sans
  • font-serif
  • font-mono

Exemplo:

<p class="font-serif">Este é um exemplo de fonte com serifa</p>
<p class="font-sans">Este é um exemplo de fonte sem serifa</p>
<p class="font-mono">Este é um exemplo de fonte mono (mesma distância horizontal para todos os caracteres)</p>

Para alterar o tamanho do texto, utilize a classe text-?, sendo ?:

  • xs;
  • sm;
  • base;
  • lg;
  • xl;
  • 2xl;
  • 3xl;
  • 4xl;
  • 5xl;
  • 6xl;
  • 7xl;
  • 8xl;
  • 9xl.

Exemplo:

<p class="text-xs">Um exemplo de tamanho de texto</p>
<p class="text-base">Um exemplo de tamanho de texto</p>
<p class="text-xl">Um exemplo de tamanho de texto</p>
<p class="text-5xl">Um exemplo de tamanho de texto</p>

Para destacar o texto:

  • italic para itálico;
  • font-bold para negrito.

Exemplo:

<p class="italic">Este texto está em itálico</p>
<p class="font-bold">Este texto está em negrito</p>
<p class="italic font-bold">Este texto está em itálico e em negrito</p>

Para controlar o alinhamento do texto, utilize:

  • text-left para alinhar à esquerda;
  • text-center para centralizar o texto;
  • text-right para alinhar à direita;
  • text-justify para centralizar.

Exemplo:

<p class="text-left">Onde estou?</p>
<p class="text-right">Onde estou?</p>
<p class="text-center">Onde estou?</p>
<p class="text-justify">Onde estou?</p>

Cores

Há uma série de cores pré-definidas no Tailwind. Essas cores são referenciadas por uma tonalidade base e um número que representa o brilho dessa cor (quanto menor o valor, maior o brilho). O brilho se inicia em 50 (mais claro), sendo o próximo valor o 100. A partir do 100, incrementa em 100 unidades até chegar ao 900. O 950 representa o brilho mais escuro.

Existem diversas tonalidades, como red, blue, emerald, etc. Para visualizar essas tonalidades, consulte o site.

Para alterar a cor do texto, utilize: text-tonalidade-brilho.

Há também as cores fixas: white e black.

Exemplo:

<p class="text-red-300">Exemplo de cor de texto</p>
<p class="text-orange-400">Exemplo de cor de texto</p>
<p class="text-emerald-500">Exemplo de cor de texto</p>
<p class="text-fuchsia-600">Exemplo de cor de texto</p>
<p class="text-violet-700">Exemplo de cor de texto</p>
<p class="text-blue-800">Exemplo de cor de texto</p>

Para alterar a cor de fundo, usamos bg-tonalidade-brilho.

Exemplo:

<body>
    <div class="bg-green-300">
        <p>Olá</p>
    </div>
    <div class="bg-red-800">
        <p class="text-white">Mundo</p>
    </div>
</body>

Margens

Para controlar as margens, usamos normalmente uma palavra indicando a margem desejada, seguida do tamanho. Para indicar a margem, utilizamos:

  • m: para todas as margens;
  • mx: para as margens na horizontal (esquerda e direita);
  • my: para as margens na vertical (superior e inferior);
  • mt: para a margem superior;
  • mb: para a margem inferior;
  • ml: para a margem esquerda;
  • mr: para a margem direita.

Em relação ao tamanho da margem, após o identificar, usamos um qualificador:

  • 0: para 0px;
  • px: para 1px;
  • 1: para 4px;
  • 2: para 8px;
  • auto: para automático.

Exemplo:

<div class="bg-green-300 m-4">
  <p class="ml-8">Olá</p>
</div>
<div class="bg-red-800 m-0">
  <p class="text-white mt-10">Mundo</p>
</div>

Espaçamento

O espaçamento (padding) define um espaçamento interno dos elementos. Ele segue as mesmas regras das margens, mas seu identificador deve ser o p ao invés do m.

Exemplo:

<div class="bg-green-300 p-4">
  <p class="pl-8">Olá</p>
</div>
<div class="bg-red-800 p-0">
  <p class="text-white pt-10">Mundo</p>
</div>

Bordas

Para definir o nível de arredondamento das bordas de um elemento, usamos a classe rounded, seguido de um quantificador:

  • rounded-none: sem borda arredondada;
  • rounded-sm: 2 pixels de arredondamento;
  • rounded: 4 pixels de arredondamento;
  • rounded-md: 6 pixels de arredondamento;
  • rounded-lg: 8 pixels de arredondamento;
  • rounded-xl: 12 pixels de arredondamento;
  • rounded-2xl: 16 pixels de arredondamento;
  • rounded-3xl: 24 pixels de arredondamento;
  • rounded-full: 9999 pixels de arredondamento.

Exemplo:

<button class="py-2 px-4 m-8 bg-yellow-400 text-black rounded-none">Salvar</button>
<button class="py-2 px-4 m-8 bg-yellow-400 text-black rounded-sm">Salvar</button>
<button class="py-2 px-4 m-8 bg-yellow-400 text-black rounded-md">Salvar</button>
<button class="py-2 px-4 m-8 bg-yellow-400 text-black rounded-lg">Salvar</button>
<button class="py-2 px-4 m-8 bg-yellow-400 text-black rounded-xl">Salvar</button>
<button class="py-2 px-4 m-8 bg-yellow-400 text-black rounded-2xl">Salvar</button>
<button class="py-2 px-4 m-8 bg-yellow-400 text-black rounded-3xl">Salvar</button>
<button class="py-2 px-4 m-8 bg-yellow-400 text-black rounded-full">Salvar</button>

Também é possível definir o arredondamento para bordas específicas, utilizando:

  • -tl: superior esquerda;
  • -tr: superior direita;
  • -bl: inferior esquerda;
  • -br: inferior direita;
  • -t: superior esquerda e superior direita;
  • -b: inferior esquerda e inferior direita;
  • -l: superior esquerda e inferior esquerda;
  • -r: superior direita e inferior direita;

Exemplo:

<button class="py-2 px-4 m-8 bg-emerald-400 text-black rounded-l-xl">Salvar</button>
<button class="py-2 px-4 m-8 bg-emerald-400 text-black rounded-l-md rounded-r-3xl">Salvar</button>
<button class="py-2 px-4 m-8 bg-emerald-400 text-black rounded-tl-3xl rounded-br-3xl">Salvar</button>

A largura da borda é definida com a classe border seguida do tamanho, em pixels, podendo ser: 0, 2, 4, 8. Caso utilize apenas border, a borda terá 1 pixel.

Exemplo:

<body class="bg-black">
  <button class="py-2 px-4 m-8 bg-rose-600 text-white border-0">Cancelar</button>
  <button class="py-2 px-4 m-8 bg-rose-600 text-white border">Cancelar</button>
  <button class="py-2 px-4 m-8 bg-rose-600 text-white border-2">Cancelar</button>
  <button class="py-2 px-4 m-8 bg-rose-600 text-white border-4">Cancelar</button>
  <button class="py-2 px-4 m-8 bg-rose-600 text-white border-8">Cancelar</button>
</body>

Também é possível definir apenas bordas específicas, utilizando:

  • -l: para a borda esquerda;
  • -r: para a borda direita;
  • -t: para a borda superior;
  • -b: para a borda inferior;
  • -x: para as bordas esquerda e direita;
  • -y: para as bordas superior e inferior.

Exemplo:

<body class="bg-black">
  <button class="py-2 px-4 m-8 bg-rose-600 text-white border-l-8">Cancelar</button>
  <button class="py-2 px-4 m-8 bg-rose-600 text-white border-r-8">Cancelar</button>
  <button class="py-2 px-4 m-8 bg-rose-600 text-white border-t-8">Cancelar</button>
  <button class="py-2 px-4 m-8 bg-rose-600 text-white border-b-8">Cancelar</button>
  <button class="py-2 px-4 m-8 bg-rose-600 text-white border-x-8">Cancelar</button>
  <button class="py-2 px-4 m-8 bg-rose-600 text-white border-y-8">Cancelar</button>
</body>

Para alterar a cor de uma borda, utilize além da classe já utilizada, a classe border-{color}, onde {color} representa a cor desejada.

Exemplo:

<body>
  <button class="py-2 px-4 m-8 bg-slate-600 text-white border-4 border-red-500">Cancelar</button>
  <button class="py-2 px-4 m-8 bg-slate-600 text-white border-x-4 border-yellow-400">Cancelar</button>
  <button class="py-2 px-4 m-8 bg-slate-600 text-white border-y-8 border-green-500">Cancelar</button>
  <button class="py-2 px-4 m-8 bg-slate-600 text-white border-t-4 border-blue-300">Cancelar</button>
</body>

Para definir o estilo da borda, utilize a classe border seguida de:

  • -solid: para borda contínua;
  • -dashed: para borda serrilhada;
  • -dotted: para borda pontilhada;
  • -double: para borda dupla;
  • -hidden: para borda escondida;
  • -none: para não utilizar borda.

Exemplo:

<body class="bg-red-400">
  <button class="py-2 px-4 m-8 bg-neutral-300 text-black border-4 border-black border-solid">Limpar</button>
  <button class="py-2 px-4 m-8 bg-neutral-300 text-black border-4 border-black border-dashed">Limpar</button>
  <button class="py-2 px-4 m-8 bg-neutral-300 text-black border-4 border-black border-dotted">Limpar</button>
  <button class="py-2 px-4 m-8 bg-neutral-300 text-black border-4 border-black border-double">Limpar</button>
  <button class="py-2 px-4 m-8 bg-neutral-300 text-black border-4 border-black border-hidden">Limpar</button>
  <button class="py-2 px-4 m-8 bg-neutral-300 text-black border-4 border-black border-none">Limpar</button>
</body>

Estados

Toda classe pode ser utiliza com uma outra classe condicional, indicando que aquela classe só será ativada se a condição for verdadeira. Alguns exemplo são:

  • hover:: indica quando o ponteiro do mouse está sobre o componente;
  • focus:: indica quando um elemento possui o foco;
  • active:: indica quando o elemento está ativo.

Exemplo:

<button class="px-4 py-2 bg-green-200 text-black rounded-md hover:bg-green-400 active:bg-green-600">Salvar</button>

Também há classes condicionais para serem utilizadas com formulários, como:

  • disabled:: define características para elementos marcados como disabled;
  • invalid: define características para elementos inválidos de acordo com seu requisitos.

Exemplo:

<form>
  <input type="text" class="bg-slate-200 border border-black disabled:cursor-not-allowed" disabled/>
  <input type="email" class="bg-slate-200 border border-black invalid:bg-red-200" />
  <button type="submit" class="px-4 py-2 bg-slate-400 rounded-lg">Enviar</button>
</form>

Modo escuro

O modo escuro também funciona como uma classe condicional, utilizando dark: antes de uma classe para designar classes que serão utilizadas quando o modo escuro estiver habilitado.

Exemplo:

<body class="bg-neutral-100 dark:bg-neutral-800">
  <p class="text-black dark:text-white">Olá, bem vindo ao meu site.</p>
  <button class="bg-blue-700 dark:bg-blue-300 text-white dark:text-black px-4 py-2 rounded-lg">Continuar</button>
</body>

Por padrão, o Tailwind obtém essa informação do sistema operacional através do recurso prefers-color-scheme. Caso seu sistema operacional não tenha esse recurso, é possível simular seu valor no inspetor de elementos do navegador Firefox.

Caso não queira trabalhar com a detecção automática do tema, é possível fazer essa alteração manualmente. Edite o arquivo tailwind.config.js para o seguinte código:

/** @type {import('tailwindcss').Config} */
module.exports = {
  content: ["./src/*.{html, js}"],
  theme: {
    extend: {},
  },
  plugins: [],
  darkMode: 'class'
}

A partir de agora, o tema será sempre o claro. Caso queira usar o tema escuro, adicione a classe dark em um item anterior na árvore de elementos da página. Exemplo:

<!doctype html>
<html id="html">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <link href="../dist/output.css" rel="stylesheet">
</head>
<body class="bg-neutral-100 dark:bg-neutral-800">
  <p class="text-black dark:text-white">Olá, bem vindo ao meu site.</p>
  <button class="bg-blue-700 dark:bg-blue-300 text-white dark:text-black px-4 py-2 rounded-lg">Continuar</button>
  <button class="bg-purple-700 dark:bg-purple-300 text-white dark:text-black px-4 py-2 rounded-lg" onclick="changeTheme()">Alterar tema</button>
</body>

<script>
  function changeTheme() {
    let html = document.getElementById('html');
    if(html.classList.contains('dark'))
      html.classList.remove('dark');
    else
      html.classList.add('dark');
  }
</script>
</html>

Responsividade

Todas as classes utilitárias do Tailwind também podem ser aplicadas de acordo com o tamanho da tela em que a página está sendo exibida. As classes a serem utilizadas para diferenciar tamanho de tela são:

  • sm: para telas de largura superior ou igual a 640 pixels;
  • md: para telas de largura superior ou igual a 768 pixels;
  • lg: para telas de largura superior ou igual a 1024 pixels;
  • xl: para telas de largura superior ou igual a 1280 pixels;
  • 2xl: para telas de largura superior ou igual a 1536 pixels.

Exemplo:

<body class="bg-red-500 sm:bg-orange-500 md:bg-yellow-500 lg:bg-emerald-500 xl:bg-blue-500 2xl:bg-purple-500">
  A cor do fundo da página vai mudar de acordo com o tamanho da tela
</body>

E possível usar as classes visible e invisible para exibir elementos de acordo com o tamanho da tela:

<body>
  <div class="text-blue-600">Eu vou sempre ser exibido, independente do tamanho da tela</div>
  <div class="text-green-600 invisible md:visible">Eu vou ser exibido apenas em telas médias para cima</div>
  <div class="text-purple-600 visible md:invisible">Eu vou ser exibido de telas pequenas até telas médias</div>
</body>

Classes personalizadas

Em algumas situações (botões, por exemplo), repetimos muito uma certa combinação de classes. Para esses casos, podemos criar componentes com a diretiva @layer, para facilitar a manutenção ao se precisar trocar um valor para todas as instâncias.

No arquivo estilo.css, adicione o seguinte código no final do conteúdo:

@layer components {
    .btn-success {
        @apply bg-emerald-500 hover:bg-emerald-600 active:bg-emerald-700 text-white font-bold py-2 px-4 rounded-lg;
    }

    .btn-failure {
        @apply bg-red-500 hover:bg-red-600 active:bg-red-700 text-white font-bold py-2 px-4 rounded-lg;
    }
}

O código anterior introduz duas classes novas, btn-sucess e btn-failure, que podem ser usadas no restante do código. Por exemplo, em index.html:

<button class="btn-success">Salvar</button>
<button class="btn-failure">Cancelar</button>

Layout

Use um elemento com a classe container para que ele tenha a sua largura máxima como a largura mínima de acordo com o tamanho de tela (sm, md, lg, etc.). Para centralizar, utilize a classe mx-auto.

Exemplo:

<div class="container bg-lime-300">
  Eu não estou centralizado
</div>

<div class="container bg-teal-300 mx-auto">
  Eu estou centralizado
</div>

Para posicionar itens dentro do espaço disponível de um container, utilizamos a propriedade flex. Podemos utilizar a classe flex-row para indicar que os elementos sejam dispostos em linha, ou flex-col para indicar que eles devem estar dispostos como uma coluna. Também é possível utilizar -reverse com as duas classes anteriores para inverter a ordem de exibição dos elementos.

Exemplo:

<div class="container bg-sky-200 mx-auto mb-4">
    <div class="flex flex-row">
      <div class="bg-purple-200 text-center">Elemento 1</div>
      <div class="bg-amber-200 text-center">Elemento 2</div>
      <div class="bg-emerald-200 text-center">Elemento 3</div>
    </div>
  </div>

  <div class="container bg-sky-200 mx-auto mb-4">
    <div class="flex flex-col">
      <div class="bg-purple-200 text-center">Elemento 1</div>
      <div class="bg-amber-200 text-center">Elemento 2</div>
      <div class="bg-emerald-200 text-center">Elemento 3</div>
    </div>
  </div>

  <div class="container bg-sky-200 mx-auto mb-4">
    <div class="flex flex-row-reverse">
      <div class="bg-purple-200 text-center">Elemento 1</div>
      <div class="bg-amber-200 text-center">Elemento 2</div>
      <div class="bg-emerald-200 text-center">Elemento 3</div>
    </div>
  </div>

  <div class="container bg-sky-200 mx-auto">
    <div class="flex flex-col-reverse">
      <div class="bg-purple-200 text-center">Elemento 1</div>
      <div class="bg-amber-200 text-center">Elemento 2</div>
      <div class="bg-emerald-200 text-center">Elemento 3</div>
    </div>
  </div>

Para informar a largura inicial de um elemento dentro de um flex, utilizamos as classes basis-{tamanho}. Por exemplo, basis-1 é referente a 4 pixels, enquanto basis-2 é referente a 8 pixels. No entanto, normalmente utilizamos flex com conteúdo cuja largura se altera, e podemos utilizar classes que usam a porcentagem do tamanho:

  • basis-1/2: 50% da largura;
  • basis-1/3: 33% da largura;
  • basis-2/3: 66% da largura;
  • basis-1/4: 25% da largura;
  • basis-3/4: 75% da largura;
  • basis-full: 100% da largura;
  • Entre outros.

Exemplo:

<div class="bg-sky-200 mx-auto mb-4">
  <div class="flex flex-row">
    <div class="bg-purple-200 text-center basis-1/4">Elemento 1</div>
    <div class="bg-amber-200 text-center basis-1/4">Elemento 2</div>
    <div class="bg-emerald-200 text-center basis-2/4">Elemento 3</div>
  </div>
</div>

Para criar elementos com uma quantidade fixa de items dispostos em uma linha, utilizamos a classe grid, seguida de uma classe grid-cols-{n}, em que n varia de 1 até 12. Por exemplo:

<div class="container mx-auto mb-4">
  <div class="grid grid-cols-4">
    <div class="bg-cyan-600 text-white text-center">Elemento 1</div>
    <div class="bg-orange-600 text-white text-center">Elemento 2</div>
    <div class="bg-lime-600 text-white text-center">Elemento 3</div>
    <div class="bg-pink-600 text-white text-center">Elemento 4</div>
  </div>
</div>

Para definir a quantidade de células que um elemento ocupa, podemos utilizar a classe col-span-{n}, em que n varia de 1 até 12. Por exemplo:

<div class="container mx-auto mb-4">
  <div class="grid grid-cols-12">
    <div class="bg-cyan-600 text-white text-center col-span-1">ID</div>
    <div class="bg-orange-600 text-white text-left pl-2 col-span-9">Nome</div>
    <div class="bg-lime-600 text-white text-center col-span-2">E-mail</div>
  </div>
</div>

Componentes prontos

O Tailwind é apenas uma biblioteca de estilização, que não possui componentes prontos. Algumas bibliotecas de componente prontos utilizam o Tailwind como base. Algumas das mais famosas são:

  • TailwindUI
  • Flowbite
  • daisyUI
  • TW Elements
